# **App Name**: WealthFront Navigator

## Core Features:

- Navigation Bar: Fixed header with company logo and menu items: Home, About Us, Services, Portfolio Review, Blog, Contact. It provides smooth scroll navigation.
- Hero Section: Large welcome banner with a headline about wealth creation and financial goals, brief company description, and a prominent CTA button.
- Statistics Counter Section: Display key metrics (number of families served, total investors, AUM, years of experience) with animated counters on scroll.
- Services Section: Grid layout of service cards for Mutual Funds, Life Insurance, NPS, Portfolio Management, Equity Trading, and Financial Planning, each with an icon, title, description, and 'Read More' link.
- Investment Comparison Calculator: Interactive tool to compare different investment options (Savings Account, Fixed Deposit, Gold, Stock Market, Mutual Funds) over 10 years, with a visual representation of amounts displayed and an input field for investment amount.
- Portfolio Evaluation Tool: CAS upload functionality to generate portfolio analysis and display the result to the user.
- Blog/Articles Section: Latest 3 blog posts in card format, showing thumbnail image, title, excerpt, and 'Read More' link. Topics include SIP vs Lumpsum, Debt Funds, Market Analysis, etc.

## Style Guidelines:

- Primary color: Navy Blue (#1a237e) provides a sophisticated feel.
- Background color: Cream (#fafaf8), slightly desaturated from the primary hue, is used for the background, ensuring readability and a clean look.
- Accent color: Gold (#ffc107), analogous to navy but with significantly different brightness and saturation, is used for call-to-action elements and highlights.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and short amounts of body text, paired with 'Inter' (sans-serif) for longer body text, to ensure a balance of modernity and readability.
- Code font: 'Source Code Pro' for displaying code snippets.
- Smooth scroll behavior and fade-in animations on scroll to provide a modern user experience.
